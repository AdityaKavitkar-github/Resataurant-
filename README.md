# Restaurant Website (HTML & CSS)

This is a self-made, fully responsive Restaurant Website built using HTML5 and CSS3. The project showcases a modern restaurant landing page with a clean UI and smooth layout that adapts to different screen sizes without using any JavaScript or frameworks.

* Technologies Used:<br>
-HTML5<br>
-CSS3<br>
-Flexbox<br>
-CSS Grid<br>
-Media Queries<br>
-Google Fonts<br>
-Font Awesome Icons<br>

* Purpose of the Project:<br>
This project was created to:<br>

-Practice responsive web design<br>
-Improve layout skills using Flexbox and Grid<br>
-Build a real-world UI using pure HTML & CSS<br>
-Showcase frontend skills for portfolio and GitHub<br>
